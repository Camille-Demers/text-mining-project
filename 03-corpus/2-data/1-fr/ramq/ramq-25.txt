Nous rendons notre site accessible aux personnes handicapées ou qui éprouvent des difficultés, selon le Standard sur l’accessibilité des sites Web du Gouvernement du Québec . L’accessibilité du Web permet aussi : Aux personnes blessées de naviguer sans utiliser la souris Aux personnes daltoniennes de distinguer les couleurs Aux personnes qui maîtrisent peu le français de mieux comprendre les textes Aux personnes âgées d’utiliser facilement le site Signification des icônes Le présent site Web contient des icônes pour simplifier le repérage de l’information et la compréhension. Voici leur signification : Contenu qui s’ouvre dans une nouvelle fenêtre Document Informations complémentaires Icônes spécifiques à la section Professionnels Régler le contraste Contraste normal Contraste élevé Contraste inversé Dépliant Document Document PDF Formulaire Formulaire de courriel sécurisé Service en ligne Site Web Téléphone Télécopieur Texte en petits caractères (taille par défaut) Texte en caractères moyens Texte en gros caractères Recherche dans le site Vérification de l’accessibilité Les outils informatiques suivants ont servi à vérifier l’accessibilité du site : NonVisual Desktop Acces ( NVDA ) , version 2019.1.1 Mozilla Firefox, version 72.0.2 Certains éléments peuvent ne pas respecter le standard d’accessibilité, en partie ou en totalité : L’abréviation de la RAMQ (il s’agit de l’appellation de la Régie de l’assurance maladie du Québec) L’outil de recherche des points de service Les documents multimédias Les documents téléchargeables Les services en ligne Les cartes dynamiques Les tableaux complexes Configuration du navigateur Le site est généralement compatible avec tous les navigateurs récents (Chrome, Firefox, Edge), peu importe la plateforme utilisée (Mac, Linux, Windows, etc.). Pour une navigation idéale, les paramètres suivants doivent être activés dans le navigateur : JavaScript Fenêtres contextuelles ( pop-up ) Fichiers de témoins ( cookies ) Lecteur PDF (exemple : Adobe Reader) Protocole SSL Assistance Si vous éprouvez des difficultés à naviguer dans notre site, à remplir un formulaire ou à accéder à certains contenus à l’aide de technologies d’adaptation informatiques, vous pouvez nous joindre pour obtenir de l’assistance.
