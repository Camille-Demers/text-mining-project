Veuillez alors l’inscrire dans le champ Code de diagnostic médical de votre facture. Vous trouverez ce champ dans la section sur les informations concernant le patient qui reçoit les services. Les codes de diagnostic proviennent de la Classification statistique internationale des maladies et des problèmes de santé connexes (CIM) de l’Organisation mondiale de la santé. La RAMQ accepte les codes de diagnostic de la 9e et de la 10e révision de la CIM (CIM-9 et CIM-10). Suivi des maladies grâce aux codes de diagnostic Le saviez-vous? Certains organismes utilisent les codes de diagnostic que vous inscrivez sur votre facture pour suivre diverses maladies, par exemple le diabète, les troubles mentaux et la maladie de Lyme. En indiquant le code de diagnostic sur la facture, vous les aidez notamment à améliorer les soins de santé et à mieux prévenir certaines maladies. Les répertoires suivants vous permettent de chercher un diagnostic selon différentes versions disponibles. Dans ce cas, vous devez : consulter la Liste des codes de diagnostic (CIM-9 – Dentisterie) ; y trouver le code qui correspond au diagnostic que vous avez rendu (p. ex. odontalgie); inscrire le code dans le champ prévu à cet effet dans votre logiciel de facturation ou FacturActe ; au besoin, consulter le document Renseignements complémentaires sur la liste des codes de diagnostic pour des précisions sur l’utilisation de la liste. Dans la CIM-9 – Dentisterie, un même code peut être lié à plusieurs diagnostics. Par exemple, si la CIM-9 indique seulement Autres affectations intéressant les dents et le parodonte – sans précision pour le code 5259, la CIM-9 – Dentisterie indique 13 possibilités, parmi lesquelles : algies dentaires; dent(s) (tissus dentaires durs) mobile(s); douleur dentaire; odontalgie.
